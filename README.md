## Español

## Descripción

**Python Learning Basics** es un repositorio de aprendizaje creado como primer acercamiento al lenguaje de programación Python. Su objetivo es reunir ejercicios básicos para practicar la sintaxis del lenguaje, comprender sus fundamentos y fortalecer la lógica de programación.

Este repositorio no está pensado como una aplicación final, sino como un espacio de práctica académica y personal donde se exploran conceptos esenciales de Python mediante ejercicios sencillos.

## Objetivo del repositorio

El propósito principal de este proyecto es mejorar el dominio inicial de Python, practicando la sintaxis y los elementos básicos necesarios para avanzar hacia proyectos más completos.

## Temas practicados

Este repositorio está orientado al aprendizaje de conceptos como:

- Sintaxis básica de Python
- Variables
- Tipos de datos
- Operadores
- Condicionales
- Ciclos
- Funciones
- Listas
- Diccionarios
- Entrada y salida por consola
- Ejercicios de lógica de programación
- Organización básica de código

## Tecnologías utilizadas

- Python
- Editor de código, como Visual Studio Code
- Terminal o consola

## Cómo usar este repositorio

### 1. Clonar el repositorio

```bash
git clone https://github.com/Sxl07/Python-Learning-Basics.git
```

### 2. Entrar al proyecto

```bash
cd Python-Learning-Basics
```

### 3. Ejecutar un archivo de práctica

Cada archivo puede ejecutarse de forma independiente con Python:

```bash
python nombre-del-archivo.py
```

En algunos sistemas también puede usarse:

```bash
python3 nombre-del-archivo.py
```

## Requisitos

Para ejecutar los ejercicios solo se necesita tener Python instalado.

Puedes verificar la versión instalada con:

```bash
python --version
```

O:

```bash
python3 --version
```

## Estado del proyecto

Este repositorio corresponde a una etapa inicial de aprendizaje. Su contenido está enfocado en practicar fundamentos del lenguaje y mejorar progresivamente la comprensión de la programación con Python.

## Notas

- Los ejercicios están orientados a práctica y aprendizaje.
- El código puede contener soluciones sencillas propias de una etapa inicial.
- El repositorio sirve como evidencia del proceso de formación y evolución en programación.
- Es una base para avanzar hacia temas como programación orientada a objetos, automatización, análisis de datos, desarrollo backend y proyectos más completos con Python.
